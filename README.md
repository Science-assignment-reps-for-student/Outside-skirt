# Outside Skirt
## How to run
```bash
$ git clone https://github.com/Science-assignment-reps-for-student/Outside-skirt.git
$ docker-compose up
```

## How to set 
```bash
$ ELASTIC_USERNAME=YourElasticUsername
$ ELASTIC_PASSWORD=YourElasticPassword

$ vim .env
ELK_VERSION=7.7.1
ELASTIC_PASSWORD=YourElasticPassword
```
